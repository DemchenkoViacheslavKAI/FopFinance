# FopFinance

Програмний засіб для обліку фінансів ФОП (офлайн, Windows desktop).

## 1. Що це за проєкт
FopFinance — це WinForms застосунок на .NET 8 з інтерфейсом на WebView2 (HTML/JS), який покриває базовий облік доходів і витрат ФОП 1-3 групи без інтеграції з онлайн-сервісами.

## 2. Технічний стек
- Backend: C# (.NET 8, WinForms)
- UI: WebView2 + HTML/CSS/JavaScript
- Зберігання: JSON/XML файли
- Логи: файл у `%LocalAppData%/FopFinance/logs/`

## 3. Запуск
### Вимоги
- Windows 10/11
- .NET SDK 8.0+
- WebView2 Runtime (зазвичай вже встановлений у Windows)

### Команди
```powershell
dotnet restore
dotnet build
dotnet run
```

## 4. Реалізований функціонал
### Доходи
- додавання
- редагування
- видалення
- перегляд у списку

### Витрати
- додавання
- прив'язка до категорії
- редагування
- видалення
- перегляд у списку

### Категорії витрат
- створення
- редагування
- видалення (з блокуванням, якщо категорія використовується у витратах)

### Звіти
- формування звіту за період
- підсумок доходів
- підсумок витрат
- чистий прибуток
- експорт звіту у JSON/XML

### Дані
- збереження всієї бази в JSON/XML
- завантаження бази з JSON/XML

### Валідація
- сума > 0
- дата обов'язкова та не в майбутньому
- обов'язковість ключових полів (джерело доходу, категорія витрати, назва категорії)

## 5. Архітектура (коротко)
- Program/MainForm: старт застосунку, ініціалізація WebView2
- BridgeService: межа між JS та C# (виклики CRUD/звіти/імпорт-експорт)
- FinanceManager: основна бізнес-логіка, агрегації, звіт за період
- Models: доменні сутності (Income, Expense, Category, Entrepreneur, Report)
- DataStorage: читання/запис JSON/XML
- wwwroot: інтерфейс і клієнтська логіка
